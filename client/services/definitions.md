# SafeMotion Definitions

## Pose / Action IDs

- `leftHandUp` = Left hand up
- `rightHandUp` = Right hand up
- `bothHandUp` = Both hands up
- `bothHandsDown` = Both hands down

## Visibility / Body-Part Requirement IDs

- `upperBody` = Upper body
- `lowerBody` = Lower body
- `head` = Head

## Body Parts Used in `Posture`

- `nose` = Nose
- `leftEye` / `rightEye` = Left/Right eye (inner, mid, outer)
- `leftEar` / `rightEar` = Left/Right ear
- `mouth.left` / `mouth.right` = Left/Right mouth corner
- `shoulder.left` / `shoulder.right` = Left/Right shoulder
- `elbow.left` / `elbow.right` = Left/Right elbow
- `wrist.left` / `wrist.right` = Left/Right wrist
- `finger.pinky.left` / `finger.pinky.right` = Left/Right pinky
- `finger.index.left` / `finger.index.right` = Left/Right index finger
- `finger.thumb.left` / `finger.thumb.right` = Left/Right thumb
- `hip.left` / `hip.right` = Left/Right hip
- `knee.left` / `knee.right` = Left/Right knee
- `ankle.left` / `ankle.right` = Left/Right ankle
- `heel.left` / `heel.right` = Left/Right heel
- `foot.left` / `foot.right` = Left/Right foot index
